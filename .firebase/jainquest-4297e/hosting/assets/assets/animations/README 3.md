# Mascot Animations

This folder should contain Lottie animation files for the mascot:

- `mascot_idle.json` - Breathing/blink loop animation
- `mascot_correct.json` - Cheer animation for correct answers
- `mascot_wrong.json` - Gentle disappointed animation (not shaming)
- `mascot_streak.json` - Celebration animation for streaks
- `mascot_complete.json` - Victory animation for lesson completion

## Design Guidelines

The mascot should be:
- Duolingo-like friendly character (but not a clone)
- Approachable and encouraging
- Modern and clean design
- Works well at various sizes (80px - 120px)

Preferred format: Lottie JSON
Alternative: Rive animations
